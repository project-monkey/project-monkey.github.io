解决步骤：

我们考虑，是不是路径配置问题

首先查看mysql版本号mysql --version，返回的版本是5.7，显然与我们的8.0不一致

所以，我们查看一下mysql的路径，在服务器端分别输入where is mysql 和 which mysql

![image-20230527155927226](/Users/tianhaoran/Library/Application Support/typora-user-images/image-20230527155927226.png)

可以看到，我们在使用的是第一个mysql

之后，我们在本地通过已经连接的mysql，查询版本号和mysql的位置，分别输入SHOW VERSION()和SELECT @@datadir;
