C++ lambda 表达式

lambda表达式是什么？

lambda表达式是一种匿名函数

举例说明

其中的[&](int)->int{}是具体的lambda表达式定义

  • [&] 表示引用捕获的外部变量
  • (int) 该函数接受一个int类型的参数
  • ->int表示该函数的返回值类型为int
  • {}函数体，表示该表达式要执行的代码

整句话定义了一个名为dp的变量，它是一个函数对象，接受一个整数类型的参数，返回整数类型
