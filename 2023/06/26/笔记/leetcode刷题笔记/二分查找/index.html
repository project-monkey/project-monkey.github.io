81.搜索旋转排序数组 II

![image-20230626170103300](/Users/tianhaoran/Library/Application Support/typora-user-images/image-20230626170103300.png)

一个不降序，但可能包含重复元素的序列中，经过一次旋转，怎么找到旋转点？

对于旋转后的序列，大小关系应该是这样的：[中，大],[小，中]

  • 如果不包含重复元素，则很容易，直接使用二分查找
  • 查找时的主要思路是判断当前的二分点(mid)切分之后的序列中，哪个序列是有序的，哪个序列是无序的，切分点pos就在那个无序的序列中
  • 对于mid切分开的两个序列[left,mid-1]和[mid+1,right]
    • 如果nums[mid]<nums[right]，则mid位于[小，中]，即第二个区间，pos在mid的左侧
    • 同理，如果nums[mid]>nums[left]，则mid位于[中，大]，即第一个区间，pos在mid的右侧
    • 但是，对于可能重复的序列，有可能会出现：nums[left] = nums[right] = nums[mid]的情况，比如3 3 3 3 0 3 3 3

一旦序列中包含了重复的元素，就无法判断哪个序列是有序的，如果遇到这种情况，就让左边界+1，右边界-1，直到出现不同的元素为止

k7SPT9TnZ6
